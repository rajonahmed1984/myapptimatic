# Project-Specific User Profile Management Guide

## Overview

Project-specific (client_project) users can manage their account through the **Profile Settings** page. This guide covers all profile management features available to project-specific users.

## Accessing Profile Settings

### Method 1: From Sidebar
1. Log in to the client portal
2. Scroll to the bottom of the left sidebar
3. Click **Profile Settings**

### Method 2: Direct URL
Navigate directly to: `/client/profile`

## Profile Management Features

### 1. View/Update Basic Information

**Available Fields:**
- **Full Name** - Your display name (updates across the system)
- **Email** - Your contact email address (must be unique)

**How to Update:**
1. Navigate to Profile Settings
2. Edit your name and/or email in the form
3. Click **Save Profile**

**Important:** If you're linked to a customer account, your name and email will also update the customer record.

---

### 2. Change Password

**Password Reset Process:**

1. Navigate to Profile Settings
2. In the **Current password** field, enter your existing password
3. In the **New password** field, enter your new password
4. In the **Confirm new password** field, re-enter your new password
5. Click **Save Profile**

**Password Requirements:**
- Minimum 8 characters
- Must contain uppercase and lowercase letters
- Must contain numbers
- Must contain special characters

**Important Notes:**
- Current password is required to change your password (security requirement)
- Password confirmation must match exactly
- Passwords are encrypted and cannot be recovered
- Change your password regularly for security

---

### 3. Profile Picture Management

**Upload/Change Profile Picture:**

1. Navigate to Profile Settings
2. Click the profile picture area or the upload box
3. Select an image file from your computer
4. Click **Save Profile**

**Supported Formats:**
- PNG
- JPG/JPEG

**Size Limit:** 2MB maximum

**Important Notes:**
- Your previous profile picture will be automatically deleted when you upload a new one
- The profile picture displays in various places:
  - Your user menu
  - Task activity comments
  - Project messages and communications
  - Admin interface when viewing your account

---

### 4. Password Reset (Forgot Password)

If you forget your password, you can reset it without logging in.

**Steps:**
1. Go to the login page (`/login`)
2. Click **Forgot your password?** link
3. Enter the email address associated with your account
4. Click **Send Password Reset Link**
5. Check your email (and spam folder) for the reset link
6. Click the link to create a new password
7. Enter your new password (following password requirements)
8. Log in with your new password

**Note:** Password reset links expire after a certain period for security reasons.

---

## Account Linking

### Customer Account Association

If your profile is linked to a customer account:
- Updates to your **Name** and **Email** will also update the customer record
- Profile picture changes will apply to both your user account and the customer profile
- This ensures consistency across the platform

---

## Security Best Practices

✓ **Do's:**
- Change your password regularly (every 3-6 months recommended)
- Use a strong, unique password
- Don't share your account credentials
- Update your email if your contact information changes
- Use a recent profile picture for identification

✗ **Don'ts:**
- Don't use the same password for multiple accounts
- Don't share your password with anyone, including support staff
- Don't use simple passwords (e.g., "123456" or "password")
- Don't write your password down in unsecured places
- Don't click password reset links from suspicious emails

## Troubleshooting

### Issue: "Email already in use" error
**Solution:** The email address is registered to another account. Choose a different email address or contact support if you need to transfer ownership.

### Issue: Password change fails with validation error
**Solution:** Ensure:
- Current password is correct
- New password meets all requirements (8+ chars, uppercase, lowercase, numbers, special characters)
- Password confirmation matches exactly

### Issue: Profile picture won't upload
**Solution:**
- Ensure file is PNG or JPG/JPEG format
- Check file size is under 2MB
- Try a different image file
- Clear browser cache and try again

### Issue: Password reset email not received
**Solution:**
1. Check your spam/junk folder
2. Wait a few minutes (email can take time to arrive)
3. Request a new reset link
4. Contact support if problem persists

### Issue: Can't access Profile Settings
**Possible Causes:**
- Not logged in (log in first)
- Browser cache issue (clear cache and reload)
- Session expired (log out and log in again)
