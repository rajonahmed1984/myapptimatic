## Implementation

### Using Component (Recommended)
```blade
<x-status-badge :status="$invoice->status" />
<x-status-badge :status="'paid'" label="Invoice Paid" />
```

### Using Helper in PHP
```php
use App\Support\StatusColorHelper;

$colors = StatusColorHelper::getStatusColors('paid');
// ['bg' => 'bg-emerald-100', 'text' => 'text-emerald-700', ...]

StatusColorHelper::getBgClass('overdue');      // 'bg-rose-100'
StatusColorHelper::getTextClass('active');     // 'text-emerald-700'
StatusColorHelper::getBadgeClasses('synced');  // 'bg-emerald-100 text-emerald-700'
```

### Manual HTML
```blade
@php
    use App\Support\StatusColorHelper;
    $colors = StatusColorHelper::getStatusColors($status);
@endphp

<div class="inline-flex items-center rounded-full px-3 py-1 text-xs font-semibold {{ $colors['bg'] }} {{ $colors['text'] }}">
    {{ ucfirst($status) }}
</div>
```

## Adding New Statuses

1. Open `app/Support/StatusColorHelper.php`
2. Add entry to `$statuses` array in `getStatusColors()` method:

```php
'new_status' => [
    'bg' => 'bg-emerald-100',      // Background color
    'text' => 'text-emerald-700',  // Text color
    'color' => 'emerald',          // Color name
    'dot' => 'bg-emerald-500',     // Indicator dot
    'icon' => 'â—',                 // Icon character
],
```

3. Use in templates:
```blade
<x-status-badge status="new_status" />
```
